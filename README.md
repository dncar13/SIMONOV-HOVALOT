# סימונוב הובלות

אתר נחיתה סטטי בעברית (RTL) עבור "סימונוב הובלות", ללא framework וללא backend. האתר נבנה כך שאפשר לפתוח את `index.html` ישירות בדפדפן.

## פתיחה מקומית

1. היכנסו לתיקייה שבה נמצאים הקבצים.
2. פתחו את `index.html` ישירות בדפדפן.
3. אם רוצים לבדוק קישורים יחסיים וסביבת שרת מקומית, אפשר גם להריץ שרת סטטי פשוט, אבל זה לא חובה.

## קבצים עיקריים

- `index.html` — דף הנחיתה הראשי.
- `styles.css` — כל העיצוב, הרספונסיביות וה־RTL.
- `script.js` — תפריט מובייל, אקורדיונים, גלילה חלקה ו־CTA צף.
- `privacy.html` — מדיניות פרטיות.
- `accessibility.html` — הצהרת נגישות.
- `robots.txt` — הוראות בסיסיות למנועי חיפוש.
- `sitemap.xml` — מפת אתר עם דומיין placeholder.
- `site.webmanifest` — הגדרות בסיסיות להתקנה כאתר.
- `favicon.svg` — אייקון מקורי בפורמט SVG.
- `logo.png` — הלוגו הראשי שמופיע באתר.

## איך משנים מספר טלפון

המספר מופיע בכמה מקומות:

- `index.html`
- `privacy.html`
- `accessibility.html`
- `script.js`

כדי לעדכן מספר:

1. חפשו את הערך `0509011040` עבור קישורי `tel:`.
2. חפשו את הערך `050-901-1040` עבור תצוגה על המסך.
3. אם מספר הוואטסאפ משתנה, עדכנו גם את `972509011040` בקישורי `wa.me`.

## איך משנים את הודעת הוואטסאפ

- ההודעה הכללית בקישורים נמצאת בתוך `index.html`, `privacy.html`, `accessibility.html` ו־`script.js`.
- הקבוע המרכזי ב־`script.js` הוא:
  - `WHATSAPP_DEFAULT_MESSAGE`

אם רוצים לשנות את נוסח ההודעה שנפתחת מהכפתורים באתר, עדכנו את כתובות `https://wa.me/...` ואת `WHATSAPP_DEFAULT_MESSAGE`.

## החלפת הדומיין לפני עלייה לאוויר

כרגע הוגדר דומיין placeholder:

- `https://example.com/`

לפני פרודקשן יש להחליף אותו במיקומים הבאים:

- `index.html` — `canonical`, `og:url`, `og:image`, `twitter:image`, `JSON-LD`
- `privacy.html` — `canonical`
- `accessibility.html` — `canonical`
- `robots.txt` — שורת `Sitemap`
- `sitemap.xml` — כל ערכי `<loc>`

## הערות חשובות

- האתר סטטי בלבד. אין backend, אין שליחת נתונים למסד נתונים ואין API חיצוני.
- יצירת הקשר מתבצעת דרך כפתורי טלפון ווואטסאפ.
- אם צריך חיבור אמיתי ללידים, טופס או שליחה למייל, יש להוסיף backend או שירות חיצוני בעתיד.
